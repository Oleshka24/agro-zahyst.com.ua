Плагін: спосіб доставки Justin та генерація накладних Justin.

== Description ==

Плагін додає спосіб доставки Justin до вашого магазину на WooCommerce. Можливість створювати накладні з замовлень буде додана згодом.
Список працюючих відділень оновлюється по АРІ служби доставки Justin.
Коли покупець обирає спосіб доставки Justin, плагін заміняє стандартні поля Область, Місто, Адреса на поля з dropdown-списком.

== Інструкція ==

Встановіть і активуйте плагін.
Додайте спосіб доставки Justin до потрібної зони доставки.
Вкажіть фіксовану вартість доставки (наприклад "0" для безкоштовної доставки).
Збережіть налаштування.
